### 專案結構

```shell
├── public/           # 靜態資源
├── src/              # 主要程式碼
│   ├── assets/       # 被編譯的靜態資源
│   │   ├── icons/    # 圖符檔
│   │   ├── images/   # 圖檔
│   ├── components/   # 共用元件
│   │   ├── theme/    # 全域樣式設定檔
│   ├── App.js        # 專案進入點
│   ├── index.js      # 入口文件
└── package.json      # 專案資訊、指令、套件及版本號
```

### 專案工具
- React
- Chakra-ui
- Styled-Components
- react-scroll
- gh-pages

### 開發指令
```bash
# 安裝專案
git clone https://github.com/Joy-Chang-2021/2023HexSchool01_CandidateWeb.git
# 安裝相關套件
npm instal
# 啟動專案
npm run start
```
